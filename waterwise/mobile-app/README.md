# WaterWise Android App (Java + XML)

## Core stack
- Material 3
- Navigation Component
- ViewModel + LiveData
- Room offline cache
- Retrofit REST client
- MQTT client realtime telemetry
- MPAndroidChart sensor graphs

## Screens
Splash, Onboarding, Auth, Dashboard, Pump Control, Sensor Monitoring, History, Notifications, Weather, Settings, Profile.
